Put the Android APK file here.

Default path served by the backend (when `SERVE_PUBLIC_WEBSITE=true`):
- `backend/public/apk/PublicDukan.apk`

Download URLs (served by backend/server.js):
- `/download/publicdukan.apk`
- `/publicdukan.apk`

You can override the file location with:
- `PUBLIC_APK_FILE_PATH=/absolute/path/to/PublicDukan.apk`
